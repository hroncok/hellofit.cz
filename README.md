# Hello FIT!

> Populárně naučný úvod do studia.

## Vývoj

Potřebujete gulp. Co a jak viz [gulpjs.com](http://gulpjs.com)

Při vývoje zapnětě `gulp watch` a spuste dist/index.html, případně si dopořidtě broswer-sync. 

## Deploy

Zatím ručně.

`gulp build`
